# NextJs Waitlist Template Starter

This project is meant to help developers get a head start in building the next big thing with first validating their idea with a waitlist.

## Tech Stack
- Nextjs
- Resend
- React Email
- Planetscale
- Drizzle
- Nextauth

## Getting Started

Fill in information in .env.local
```
RESEND_API_KEY=
AUTH_SECRET=
DATABASE_URL=
DATABASE_URL_TEST=
EMAIL_SERVER_USER=
EMAIL_SERVER_PASSWORD=
EMAIL_SERVER_HOST=
EMAIL_SERVER_PORT=
EMAIL_FROM=
```

First, run the development server:

```bash
bun install
bun dev
```
